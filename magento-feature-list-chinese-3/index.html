Magento 基本功能列表（翻译三）

发表于 2016-12-14 分类于 其他

架构及安全

  • 建立在现代的技术栈上，使用了流行的框架，支持行业标准
  • 可以进行细粒度的自定义，模块化的代码库，很容易添加和移除功能
  • 业务逻辑使用稳定 service contract interfaces ，以确保站点更容易升级和自定义
  • 可以使用高效的 web Api （SOAP 和 REST）与第三方应用集成
  • 通过 plug-ins 和依赖注入的软件设计模式，更顺利的管理扩展之间的交互
  • 加快上线速度，通过完全自动化的测试框架提高代码质量
  • Update code with a standalone installer that checks for prerequisites before starting the installation process
  • 通过命令行接口自动化某些开发和生产任务
  • 定期的扫描、外部渗透测试，bug 赏金程序，促进平台核心的安全
  • Magento 安全中心和安全警报邮件列表，会通知你安全问题

布局和主题

  • 自定义 blank 主题 或 luuma 主题，以加快上线速度
  • 通过主题集成，更容易地位不同的国家、品牌或季节促销活动创建站点
  • 通过 Magento UI 和 模式库，使得主题开发更快速和更一致
  • 使用 Less 预处理节省时间，通过代码复用和全局变量的更改，更易于站点维护
  • 通过分离表现层和业务逻辑，降低前段编码的复杂性

性能和可扩展性

  • 使用预先集成在 Apache 中的 Varnish 缓存页面内容，获得更好的性能
  • Implement hole punching and Edge Side Includes (ESI) to handle page elements that can’t be cached
  • 收益于更快的索引，以及增量索引，以更新站点内容和更好的性能
  • Minify JavaScript, CSS 和 HTML ，压缩图像，以减少文件大小，加快加载速度
  • 将 JavaScript 和他们的依赖打包，加快浏览器加载速度
  • 访问的第一个页面缓存所有 Js 和 CSS ，以尽量减少后续页面必须加载的内容
  • 使用 Magento 内置的性能工具包，测量和优化站点的性能
  • 允许大型团队（许多管理用户）管理订单，按计划更新产品
