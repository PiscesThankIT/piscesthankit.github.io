默认字段值

发表于 2016-08-22 分类于 Magento 2 用户使用手册

原文网址

为了在创建商品时节约时间，有些商品的字段会参考另外一些字段作为默认值。您可以直接用默认值，也可以输入其他。下面的字段是会自动生成默认值的：

字段

描述

SKU

基于商品名称(product name)

Meta Title

基于商品名称(product name)

Meta Keywords

基于商品名称(product name)

Meta Description

基于商品名称和描述(product name and description)

&#123;&#123;&#125;&#125;这种两个花括号括起来的是占位符，他表示某个字段的值。商品属性集中的任何属性代码都可以用来做占位符。

修改占位符

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)
  2. 在左侧面板目录(Catalog)下选择目录(Catalog)
  3. 展开商品字段自动生成(Product Fields Auto-Generation)，进行修改。
    举例来说，假设您想给每个商品加入某个关键字，或者给每个meta 描述加入一个短语，您可以在合适的字段内输入该关键字或短语。
  4. 完成后，点击保存设置
