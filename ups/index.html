UPS

发表于 2016-10-17 分类于 Magento 2 用户使用手册

原文地址

United Parcel Service 提供美国国内及多于220个国家的包裹运送服务。

UPS 现在采用体积重量决定某些运输费用

第一步：创建一个UPS账户

要给您的客户提供UPS快递方法，您必须首先创建一个UPS账户。

第二步：启用 UPS

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)

  2. 在左侧面板选择销售(Sales)> 送货方式(Shipping Methods)

  3. 展开 UPS(UPS) 做如下操作:

  4. 设置 启用(Enabled)为“Yes”

  5. 如果是UPS 标准账户，设置 UPS Type 为 “United Parcel Service”，做如下事情：
    a. 设置 Live Account 为以下值：

    value

    description

    Yes

    以生产模式使用UPS，对客户提供UPS快递方式

    No

    以测试模式使用UPS

    b. 在 Gateway URL 字段中填入用来计算UPS运费的URL。一般已经填好，不需要修改的。
    要生成 shipping labels，UPS Type 必须被设置为 United Parcel XML

  6. 如果是 UPS XML 账户，做如下操作：
    a. 设置 UPS Type 为 United Parcel Service XML
    b. 输入如下认证信息：

    c. 设置 Mode 为 Live，这样会以安全的连接连接到 UPS 系统。Development 模式不会通过安全的连接发送数据。
    d. 验证 Gateway XML URL
    e. 设置 Origin of the Shipment 为发货的起始位置。
    f. 如果您与UPS 达成了其他费用协议，请设置 Enable Negotiated Rates 为 Yes，然后在 Shipper Number 中输入UPS分配给你的六位数字码。

  7. 在 标题(Title)字段中，输入结算时显示的名称。默认情况下，标题是 United Parcel Service

第三步：完成集装箱描述

  1. 设置请求包装类型(Packages Request Type)

    • Use origin weight (few requests)
    • Divide to equal weight (one request)

  2. 在 Container 字段中，指定用于装运的典型包装类型

  3. 设置重量单位(Weight Unit)。UPS 支持的重量单位因不同国家而不同。如果你不是很清楚，请询问UPS。支持的重量单位有：

    • LBS
    • KGS

  4. 设置 目的地址类型(Destination Type) 为以下值：

    value

    description

    Residential 住宅类

    大多数运输是B2C

    Commercial 商业型

    B2B类型

    Define Automatically 自动判断

    UPS 自行判断类型，并使用相应的费率

  5. 输入允许的 最大包裹重量 (Maximum Package Weight)

  6. 设置 Pickup Method

    • Regular Daily Pickup
    • On Call Air
    • One Time Pickup
    • Letter Center
    • Customer Counter

  7. 设置物流公司接受的最小包裹重量

第四步：设置手续费

  1. 设置计算手续费(Calculate Handling Fee )为固定的(Fixed)或者按百分比(Percent)
  2. 设置Handling Applied(Handling Applied) 为按单收取(Per Order)或者按包裹收取(Per Package)
  3. 设置收取的手续费(Handling Fee)，输入百分数请以小数的形式。比如输入0.25来表示25%

第五步：指定允许的方法和可用的国家

  1. 在Allowed Methods列表选中您提供的UPS方法。该方法会出现在接结算时UPS 项下。（多选按住Ctrl键）

  2. 设置 Free Method，然后做如下操作：

    • 设置Free Shipping with Minimum Order Amount为 Enabled，然后输入 Minimum Order Amount for Free Shipping
    • 设置Free Shipping with Minimum Order Amount 为 Disabled

  3. 在 Displayed Error Message box中输入UPS 不可用时的错误提示信息。

  4. 设置哪些国家可用(Ship to Applicable Countries)

    name

    description

    所有允许的国家All Allowed Countries

    Allow Country设置中允许的所有国家都允许

    指定国家Specific Countries

    选择本项后，下面的(Ship to Specific Countries)列表可以用了

  5. 不可用时是否显示(Show Method if Not Applicable)可选择如下选项：

    name

    description

    Yes

    显示所有UPS的方法，不管可不可用

    No

    只显示可用的方法

  6. 要创建日志来记录UPS的详情，请设置 Debug 为 Yes

  7. 在 Sort Order中输入数字来指定 UPS 在运输方式列表中的排序。0表示列表第一个。

  8. 完成后，点击 Save Config
