可配置商品添加配置项(Adding Configurations)

发表于 2017-01-03 分类于 Magento 2 用户使用手册

原文地址

下面这个例子中添加了三个颜色三个尺码的配置项。总共会创建9个简单商品，每个商品都有唯一的 SKU ，页面顶部的进度条会显示你现在在哪里，并引导你完成所有步骤。

步骤一：选择属性

  1. 上一部分 可配置商品(Configurable Product) 点击保存后，下拉到 Configurations 部分，然后点击 Create Configurations
  2. 勾选你要包含的属性。比如，这里我们选择 color 和 size
  3. 列表显示了该属性集中所有可被应用在可配置商品上的属性
  4. 如果你需要添加新的属性，点击 Create New Attribute 完成attribute 的属性，然后点击 Save Attribute，然后勾选复选框选择属性。
  5. 右上角，点击 Next

步骤二：输入属性值

  1. 为每个属性勾选应用到该商品上的值
  2. 重新排列属性的顺序，可以拖动 图标到新的位置。他们的顺序决定着产品详情页中下拉列表的位置。
  3. 在进度条中，点击 Next

步骤三：配置图片、价格和数量

这一步决定着每个配置项的图片、价格和数量。可选项是一样的，你只能选一个。可以是 apply the same setting to all SKUs, apply a unique setting to each SKU, or skip the settings for now

  1. 选择应用的配置选项

    • 配置图片
      方法一：对所有 SKUs 应用同一组图片(Apply a Single Set of Images to All SKUs)

      1. 选择 Apply a Single Set of Images to All SKUs
      2. 浏览选择图片或者拖动图片到下图的框中

        方法二：对每个 SKU 应用单独的图片
        因为我们已经为父商品上传了图片，我们将使用这个选项来为每个颜色上传图片。假设某个客户购买了某个颜色的T恤，那么购物车中显示的就是这里的图片。
      3. 选择 Apply unique images by attribute to each SKU
      4. 选择图像要说明的属性，比如：color
      5. 对每个属性值，要么浏览选择图片，要么拖拽到框中
        If you drag the an image to a value box, it appears in the sections for the other values, as well.如果你想要删除图片，点击 图标

    • 配置价格
      方法一：对所有 SKU 应用相同的价格 (Apply the Same Price to All SKUs)

      1. 选择 Apply single price to all SKUs
      2. 输入 价格

        方法二：对每个 SKU 应用不同的价格 (Apply a Different Price for Each SKU)
      3. 如果商品的各种变体价格不一样，选择 Apply unique prices by attribute to each SKU
      4. 选择价格差异基于的属性
      5. 设置每个属性值的价格。在这个例子中，XL 尺码的更贵一点

    • 配置数量
      方法一：对所有的 SKU 应用相同的数量 (Apply the Same Quantity to All SKUs)

      1. 如果所有 SKU 的库存数量都是一样的，选择 Apply the Same Quantity to All SKUs
      2. 输入 数量

        方法二：不同的属性应用不同的数量 (Apply Different Quantity by Attribute)
      3. 如果每个属性的数量不同，选择 Apply Different Quantity by Attribute
      4. 设置每个的值
  2. 完成后，点击右上角的 Next

步骤四：生成产品配置

等一会儿，等产品列表显示出来以后，做如下事情：

  • 如果你对生成的配置满意，点击 Next
  • 要做修改，点击 Back

步骤五：添加产品图片

  1. 下拉到 Images and Videos 部分，然后展开
  2. 点击 相机 图标，选择该商品的主图
    了解更多，请查阅 Images and Video

步骤六：完成产品信息

下拉完成以下部分的信息

  • Content
  • Images and Videos
  • Related Products, Up-Sells, and Cross-Sells
  • Search Engine Optimization
  • Customizable Options
  • Products in Websites
  • Design
  • Gift Options

步骤七：发布商品

  1. 如果该商品可以发布了，则设置 Enable Product 为 Yes
  2. 你可以点击 Save & Close 或者以客户的视角浏览商品的信息

步骤八：配置购物车缩略图(可选)

如果你商品的每个变体都有不同的图片，你可以配置购物车的缩略图使用正确的图片

  1. 后台侧边栏，点击 Stores，在 Settings 下，选择 Configuration
  2. 左侧边栏 Sales 下，选择 Checkout，展开 Shopping Cart 部分
  3. 设置 Configurable Product Image 为 Product Thumbnail Itself
  4. 完成后，点击 Save Config

要点

  • 可配置产品允许客户从下拉列表中选择选项。每个选项实际上都是一个单独的简单商品
  • 每个下拉列表的值都是基于 dropdown 类型的属性。drop-down 类型的属性必须包含在属性集中，该属性集则是当前可配置产品的模板
  • 购物车中产品的缩略图可以配置为产品的图片，或者是可配置产品的变体产品的图片
