Magento 基本功能列表（翻译一）

发表于 2016-12-14 分类于 其他

本系列将概述 Magento 所有版本都具有的基本功能。

查看完整的原始英文文档，请点击这里下载（文档还包含 Magento 企业版本的功能列表）

促销和定价

  • 为特定的店铺、商品类别、客户组或者商品，设置促销价格
  • 为特定的店铺、客户组、时间段、商品、商品类别，创建灵活多样的优惠券
  • 为线下销售、电子邮件营销、资讯订阅等生成优惠券代码
  • 跟踪和报告优惠券的使用情况
  • 提供各种折扣方式：免邮、按百分比的、按固定数量的、买 X 免费送 Y
  • 根据购买量的不同，设置多级价格策略
  • 支持最低广告价格

搜索引擎优化

  • 自动生成 Google 站点地图
  • 创建搜索引擎友好的 URL
  • 通过 URL 重写完全控制 URL
  • 为商品，类别，内容页创建元数据

网站管理

  • 通过同一个后台管理多个网站和店铺。
  • 通过现代、友好的管理面板，快速学会如何管理和监控店铺的情况
  • 通过自定义管理面板，保存为不同的视图，帮助你更高效地管理商品、客户及订单
  • 定义角色和权限，控制各项管理功能的访问
  • 使用内容管理系统来创建和轻松编辑页面和内容块
  • 通过 100% 可定制的响应式参考主题，打造独一无二的站点
  • 根据地理位置、产品类型、客户组（例如，零售 vs 批发），设置税率
  • 添加验证码功能，防止自动化软件尝试登录

（商品）目录管理

  • 可创建简单商品、可配置商品（例如，选择尺寸、颜色等）、捆绑商品（例如，同样型号的电脑，可以选配不同的 CPU）、组合商品（例如，沙发套件，可以单独出售，更多的是为了方便客户的）
  • 虚拟商品
  • 包括可下载的/电子产品，可提供样本
  • 允许客户自定义的产品（例如，客户可以上传图片、设置文本，制作文化衫）
  • 通过在线引导过程创建新的产品
  • 可定义无限的产品属性
  • 通过创建预定义的产品模板，来快速设置新产品的特定属性集
  • 高效地批量导入导出目录信息
  • 轻松管理库存，支持延期交货（with support for back ordered items）
  • 需要补充库存时，会收到通知
  • 选择缺货时的通知和显示选项
  • 定义可以加入购物车的商品最少和最多数量
  • 批量更新商品，例如，为多个商品设置某个值
  • 自动调整图片大小，包括加水印
  • 如果客户搜索无结果，可以改写和重定向搜索结果
  • 设置搜索目录属性权重和搜索同义词，来管理站点的搜索转换
  • 适当的评价和产品标签

目录浏览

  • 通过层级导航，根据价格范围，品牌，颜色及其他属性，筛选目录和搜索结果中的商品
  • 商品目录的呈现可以是表格或列表方式
  • 自定义商品目录页面的外观和感觉
  • 显示商品比较的历史
  • 显示最近浏览过的商品
  • 添加面包屑导航

商品浏览

  • 每个商品可显示多张图片
  • 商品图片可以放大查看
  • 可以播放 YouTube 或 Vimeo 上的商品视频
  • 包括显示颜色、面料或其他的面板（swatches）
  • 包括组合商品的视图
  • 设置相关商品、up-sell（同一需求的更高级商品）、cross-sell（追加销售，客户的其他需求）
  • 表明库存情况
  • 显示分级价格
  • 显示商品的评价和评分
  • 可添加商品到愿望清单
