php 不是内部或外部命令，也不是可运行的程序或批处理文件

发表于 2016-08-22 分类于 Magento 2 入门
在 windows CMD 下运行php bin\magento 出现错误提示：’php’ 不是内部或外部命令，也不是可运行的程序或批处理文件，这是因为没有配置 PATH 环境变量。 以 win10 为例，找到此电脑右击选择属性，点击左侧的高级系统设置，选择高级 > 环境变量 弹出环境变量窗口，找到PATH，点击编辑 在编辑环境变量窗口，输入你的 php.exe 所在的路径，保存后要重启一下才会生效.
