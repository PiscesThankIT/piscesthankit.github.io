创建色板(Creating Swatches)

发表于 2017-08-21 分类于 Magento 2 用户使用手册

原文地址 在下面的例子中，Sylvia Capris 可供选购的有特定的红色、绿色和蓝色。因为色板取自产品图像，所以每个色板都是颜色的真实表示。color 属性用来管理所有产品的颜色和色板信息。

步骤一：创建色板

使用以下任一方法创建色板

方法一：添加一个颜色色板

  1. 使用你喜欢的图像编辑器打开产品图片，用拾色器拾取对应的颜色，记下颜色值。
  2. 后台侧边栏，点击 Stores > Attributes > Product
  3. 在列表中，找到 color ，点击进行编辑
  4. 验证 Catalog Input Type for Store Owner 被设置为 Visual Swatch
  5. 在 Manage Swatch 下点击 Add Swatch ，然后做以下事情： a. 点击新色板，弹出菜单，选择 Choose a color b. 在拾色器中输入之前取得的颜色值 c. 点击右下角的颜色图标()，保存该颜色。 d. 在 Admin 这栏输入 label 用于后台查看的标签。如果合适，给每种语言设置对应的颜色的翻译。在下面的例子中，我们在 Admin label 中包含了产品的 SKU ，因为该颜色用于特定的产品。您可以在 label 中使用空格或者下划线，但不要使用连字符。 e. Is Default 栏中，选择作为默认值的色板 f. 要改变色板的顺序，只需要简单地拖动色板到特定位置。
  6. 完成后，点击 Save Attribute，然后提示刷新缓存
  7. 最后一步就是编辑产品，更新产品的 color 属性。要批量修改产品，请参考下面的步骤。

方法二：上传图像色板

  1. 使用图像编辑器打开产品图片，选取一小块能够代表颜色、图案、纹理的区域保存。尺寸取决于 theme 。一般来说，保存成正方形有利于保持纵横比。
  2. 后台侧边栏，点击 Stores > Attributes > Product
  3. 在列表中，找到 color ，点击进行编辑
  4. 验证 Catalog Input Type for Store Owner 被设置为 Visual Swatch
  5. 在 Manage Swatch 下点击 Add Swatch ，然后做以下事情： a. 点击新色板，弹出菜单，选择 Upload a file b. 选择准备好的图片，上传 c. 重复上述步骤，给所有色板都设置好图片 d. 在 Admin 这栏输入 label 用于后台查看的标签。如果合适，给每种语言设置对应的颜色的翻译。在下面的例子中，我们在 Admin label 中包含了产品的 SKU ，因为该颜色用于特定的产品。您可以在 label 中使用空格或者下划线，但不要使用连字符。
  6. 完成后，点击 Save Attribute，然后提示刷新缓存
  7. 最后一步就是编辑产品，更新产品的 color 属性。要批量修改产品，请参考下面的步骤。

步骤二：更新产品

  1. 后台侧边栏，点击 Products > Inventory > Catalog
  2. Filter 筛选出需要修改的产品，下面的例子根据部分产品名称进行筛选。
  3. 在列表中，勾选出此次要更改的产品，在本例中，所有蓝色的紧身裤被选中，然后，在 Actions 中选择 Update Attributes
  4. 向下拉找到 color 属性，勾选 change 复选框
  5. 选择应用的色板，点击 Save 刷新缓存
