固定运费(Flat Rate Shipping)

发表于 2016-10-14 分类于 Magento 2 用户使用手册

原文地址

固定运费是预先定义的固定的收费方式，可以按件或者按次计算。固定运费是一个简单计费方案，特别适合一些采用固定运费的物流公司。

设置固定运费

  1. 后台侧边栏，点击店铺(Stores) > 设置(Settings) > 配置(Configuration)

  2. 在左侧面板选择销售(Sales)> 送货方式(Shipping Methods)

  3. 展开 固定运费(Flat Rate) 做如下操作:
    a. 设置 启用(Enabled)为“Yes”

    b. 输入标题(Title)。
    c. 输入方法名(Method Name)，他会出现在购物车计算费率边上。默认的名字叫“Fixed”。如果您收取手续费，您可以把方法名改为“Plus Handling” 或者其他合适的名字。
    d. 类型(Type)中有如下选项，描述了固定运费的使用方式：

    name

    description

    None

    固定运费会被列出来，但费率为0，相当于包邮

    Per Order 每笔订单

    按订单计算

    Per Item

    按件计算，不管是不是同一种商品，每一件都计算

    e. 在价格(Price)中输入要收取的价格
    f. 如果收取额外手续费的话，在计算手续费(Calculate Handling Fee)中选择：

    然后在手续费(Handling Fee)中输入相应的值。比如，如果费用按百分比算，输入0.06表示百分之六。如果按固定的算法，输入数值。
    g. 在出错提示信息(Displayed Error Message)中输入固定运费方式出错时的提示信息。
    h. 设置哪些国家可用(Ship to Applicable Countries)

    name

    description

    所有允许的国家All Allowed Countries

    Allow Country设置中允许的所有国家都允许

    指定国家Specific Countries

    选择本项后，下面的(Ship to Specific Countries)列表可以用了，选择哪些国家的客户可以用固定运费

    i. 在排序(Sort Order)中输入数字，来决定包邮选项在送货方式列表中顺序（0表示第一个，1表示第二个，依次类推）

  4. 完成后，点击 Save Config
