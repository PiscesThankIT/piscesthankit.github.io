表格布局

发表于 2016-08-18 分类于 Magento 2 用户使用手册

原文网址

您可以根据需要修改表格中展示的列，哪些展示哪些不展示，改变他们的顺序并保存为视图(Views)。默认情况下，显示20个中的9个（实际上是19个中的9个显示）

更改是否显示列

  1. 右上角，点击列(Columns)控件，然后可以做以下操作：
    • 要在表格中展示的打勾
    • 不要再表格中显示的去掉勾
  2. 记得下拉浏览所有可用的列

移动列

  1. 点击列的标题，按住不放
  2. 拖动到新的位置，然后松开

保存表格的视图

  1. 点击视图(View)点击保存当前视图(Save Current View)
  2. 输入视图的名称，然后点击箭头保存。当前视图名就变成您刚刚建立的那个视图了。

修改表格视图

点击视图(View)，然后做下列操作：

  • 点击视图名称切换另一个视图
  • 修改视图的名字，点击编辑(Edit)图标，进行修改。
