可下载商品(Downloadable Product)

发表于 2017-01-04 分类于 Magento 2 用户使用手册

原文地址

可下载商品可以是任何以文件形式交付的东西，比如电子书、音乐、视频、软件或更新。你可以出售专辑，单独出售每首歌曲。你还可以使用可下载商品提供电子版的产品目录。

因为直到付款后才可以下载商品，因此你可以提供样品。比如一本书的摘录、音频文件或视频文件中的一段剪辑，这样客户可以在购买前先感受下。供下载的文件可以放在自己的服务器上，也可以放在其他服务器上。

可下载商品可以配置为要求客户登陆后获取下载链接，或者可以通过邮件发送，并与其他人分享。了解更多 配置可下载商品的选项

以下将指引你创建一个基本设置的可下载商品。必填项以红色星号(*)标记出来。填完必填项并保存后，你可以继续填写剩下的字段、添加图片。

Downloadable file names can include letters and numbers. Either a dash or underscore character can be used to represent a space between words. Any invalid characters in the file name are replaced with an underscore.

第一步：选择产品类型

  1. 后台侧边栏，点击 Products ，在 Inventory 下选择 Catalog
  2. 右上角 Add Product 菜单，选择 Simple Product

第二步：选择属性集

属性集是产品的模板，要选择属性集，可以在搜索框里输入属性集的名字。

第三步：完成必要的设置

  1. 输入产品的名称 Product Name
  2. 使用基于产品名称的默认 SKU 或者重新输入一个 SKU
  3. 输入产品的价格 Price
  4. 因为产品还没有准备好发布，所以把 Enable Product 切换到 No 的位置
  5. 点击 Save
    产品被保存后，左上角会出现 Store View chooser（用来选择作用于哪个店铺）
  6. 选择作用于那个 store view

第四步：完成基本设置

  1. 设置 Tax Class 为以下值之一：
    • None
    • Taxable Goods
  2. 输入库存数量 Quantity，注意默认情况下 Stock Status 设置为 In Stock
  3. 可下载商品不用配送，也没有重量。
  4. Visibility 默认设置是 Catalog, Search
  5. 把产品分配给 Categories，点击选项框，这一步也可以创建 category ，都是显而易见的，所以不详细翻译了。
  6. 如果要标记改商品为新品，勾选 New 复选框
  7. 设置 Format 为以下值
    • Download
    • DVD
      如果必要的话，你可以编辑属性来增加更多选项值

      可能还有很多其他属性。属性集的选项是不一样的，你可以稍后再填写他们。

第五步：完成可下载商品信息

向下滚动找到 Downloadable Information，然后展开这部分，然后勾选 Is this downloadable product?

这部分有信息由两块内容组成。第一部分描述下载链接，第二部分描述样本文件。更多配置查看 configuration

完成链接部分

  1. 输入 标题 Title

  2. 如果需要的话，勾选 Links can be purchased separately

  3. 点击 Add Link，然后做如下事情：
    a. 输入标题和价格
    b. 不管是正式文件还是样品文件，选择下列方法之一

    Upload File

    上传到本服务器

    URL

    输入 URL

    c. 设置 Shareable 为下列值之一

    No

    要求客户必须登陆账户获取下载链接

    Yes

    通过邮件发送连接，这样客户很方便与其他人分享

    Use Config

    使用在 Downloadable Product Options 中指定的方法

    d. 做如下操作：

    • 限制每位客户的下载，输入 Max. Downloads
    • 无限下载，勾选 Unlimited 复选框

  4. 要增加一个链接，点击 Add Link，然后重复上述步骤

完成样品部分

  1. 输入 标题 Title

  2. 点击 Add Link

  3. 完成以下操作：
    a. 输入每个样本的标题
    b. 选择下列之一的发行方法：

    Upload File

    上传到本服务器

    URL

    输入 URL

    c. 点击 Add Link ，并重复上述步骤增加样本
    d. 拖放 图标调整样本的位置

第六步：填写商品信息

往下拉，然后根据需要填写以下部分：

  • Content
  • Images and Videos
  • Related Products, Up-Sells, and Cross-Sells
  • Search Engine Optimization
  • Customizable Options
  • Products in Websites
  • Design
  • Gift Options

第七步：发布商品

  1. 如果你已经准备好要发布产品了，那么设置 Enable Product 为 Yes
  2. 下面你可以点击右上角的 Save & Close 或者 Save 然后再 Admin 菜单中选择 Customer View 检查下产品发布信息是否 OK

要点

  • 可下载商品可以上传到当前服务器上，或者提供到另一服务器的下载链接
  • 你可以决定客户可以下载的次数
  • 购买可下载商品的客户可以被要求结账前必须登陆
  • The delivery of a downloadable product can be made when the order is in either a “Pending” or “Invoiced” state.
