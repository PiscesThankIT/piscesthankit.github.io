可配置商品(Configurable Product)

发表于 2017-01-03 分类于 Magento 2 用户使用手册

原文地址

可配置商品看起来有一系列的选项。每个选项代表着单独的具有唯一 SKU 的简单商品。这样可以追踪每个变体的库存。你可以通过简单商品的 custom option 来达到类似的效果，但是不能跟踪每个商品变体的库存。

第一步：选择产品类型

  1. 后台侧边栏，点击 Products ，在 Inventory 下选择 Catalog
  2. 右上角 Add Product 菜单，选择 Configurable Product

第二步：选择属性集

属性集决定着该产品中使用的选择字段有哪些。默认情况下，是 default 属性集。对于可配置商品来说，属性集必须包含至少一个 drop-down 属性，另外，drop-down 属性的 scope 必须是 Global 下面这个例子里使用的下拉选项式属性是 color 和 size

要选择属性集，可以在搜索框里输入属性集的名字。

第三步：完成必要的设置

  1. 输入产品的名称 Product Name
  2. 使用基于产品名称的默认 SKU 或者重新输入一个 SKU
  3. 输入产品的价格 Price
  4. 因为产品还没有准备好发布，所以把 Enable Product 切换到 No 的位置
  5. 点击 Save
    产品被保存后，左上角会出现 Store View chooser（用来选择作用于哪个店铺）
  6. 选择作用于那个 store view

第四步：完成基本设置

  1. 设置 Tax Class 为以下值之一：
    • None
    • Taxable Goods
  2. 输入库存数量 Quantity，注意默认情况下 Stock Status 设置为 In Stock
  3. 输入产品的重量
  4. Visibility 默认设置是 Catalog, Search
  5. 把产品分配给 Categories，点击选项框，这一步也可以创建 category ，都是显而易见的，所以不详细翻译了。
  6. 如果要标记改商品为新品，勾选 New 复选框
  7. 选择 Country of Manufacture

    可能还有很多其他属性。属性集的选项是不一样的，你可以稍后再填写他们。

第五步：保存并继续

现在保存你的商品，在下一个步骤中，你将为每个产品的变体配置设置。

  1. 右上角，点击 Save
  2. 下面阅读 添加配置(Adding Configurations)
