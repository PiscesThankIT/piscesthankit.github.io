控制面板

发表于 2016-08-17 分类于 Magento 2 用户使用手册

原文地址

控制面板通常是您登录管理后台后见到的第一个页面，提供您店铺中销售和客户活动信息的简要概述。左边快照是店铺开张以来所有销售额、平均每单金额、最后五笔订单以及客户搜索词。图表展示了所选时间范围内订单和销量。您可以通过图表上方的选项卡切换视图(Orders 还是 Amounts).底部的选项卡提供有关畅销商品、浏览最多商品、新客户和买的最多的客户的快报。

控制面板是默认的管理后台起始页面。不过您可以修改设置让您登录后第一个页面是别的页面。您还可以设置控制面板中报告所使用的起始日期，并禁用图表展示。

您可以为每个店铺视图(store view)生成控制面板的快照报告。页面底部的选项卡总括了指定时间段内最畅的销商品，浏览最多的商品，新的客户，以及买的最多的客户。（不要问我为什么重复了。。）

设置图表

  1. 在 Admin 主菜单中，点击店铺(Stores)，在设置(Settings)下选择设置(Configuration),之后执行下面的任一操作。

    • 启用图表

      1. 左侧面板，在高级(Advanced)之下，选择管理员(Admin)

      2. 在控制面板(Dashboard)部分，选择启用图表(Enable Charts)为是(Yes)

    • 设置起始日期

      1. 左侧面板常规(General)>报告(Reports)

      2. 在控制面板(Dashboard)部分，做如下操作：
        a. set Year-To-Date Starts to the Month and Day.
        b. Set Current Month Starts to the Day.

        （此处没看明白，各位看官有明白的望不吝赐教，补充更正。）

    • 选择数据源

      如果您要处理的数据比较多，您可以关闭展示实时数据以优化控制面板的性能。

    1. 在左侧面板中，点击销售(Sales)让他展开，再选择展开的菜单销售
    2. 点击展开控制面板(Dashboard)，做如下操作：

      • 要实时数据，设置使用聚合数据 (测试版)(Use Aggregated Data (beta))为是

      • 使用历史数据，设置使用聚合数据 (测试版)(Use Aggregated Data (beta))为否

    • 更改起始页面

      1. 左侧面板，点击展开高级(Advanced) > 管理员(Admin)

      2. 点击展开起始页(Startup Page)

      3. 选择您想要的起始页面，列表列出了后台所有的页面。

  2. 完成后，点击保存设置(Save Config)
