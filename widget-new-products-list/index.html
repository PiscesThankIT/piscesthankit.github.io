新产品列表 (New Products List)

发表于 2016-11-15 分类于 Magento 2 用户使用手册

原文地址

新产品列表是说明动态内容的好例子，他是从产品库中取出的实时的数据。默认情况下，新产品列表包含的是最新添加的八个产品。但是，也可以通过配置让他只包含某个时间段内的产品。

第一步：设置每个产品的日期范围

要让产品按照时间范围出现在新产品列表中，您必须给产品填入 “New From” 日期。

  1. 后台侧边栏，点击 Products，在 Inventory 下，选择 Catalog
  2. 选择你要更改的产品进行编辑：
    a. 在 Set Product as New From Date 字段中，点击日历小空间，然后选择开始时间
    b. 在 Set Product as New To Date 字段中，点击日历小空间，然后选择结束时间
  3. 完成后，点击 Save
  4. 系统提示你清理缓存，照做。
    现在新产品列表中就会出现在时间范围内的产品了。

第二步：创建小部件

决定新产品列表位置的代码由 Widget 工具生成。

  1. 后台侧边栏，点击 Content，在 Elements 下，选择 Widgets

  2. 右上角点击 Add New Widget

  3. 在设置部分，做如下事情：
    a. 设置 Type 为 Catalog New Products List
    b. 选择 Design Theme 为当前主题

  4. 点击 Continue

  5. 在 Storefront Properties 部分，完成下列字段：

    Widget Title

    输入描述标题，这个标题只在后台可见

    Assign to Store Views

    选择 widget 可用的 store view

    Sort Order

    （可选）输入数字决定次序（0 = 第一个，以此类推）

第三步：选择位置

  1. 在 Layout Updates 部分，点击 Add Layout Update，然后做如下事情：
    a. 设置 Display On 为 Specified Page
    b. 设置 Page 为 CMS Home Page
    c. 设置 Block Reference 为 Main Content Area
    d. 设置 Template 为下列之一：

    • New Products List Template
    • New Products Grid Template

  2. 点击 Save and Contine Edit

第四步：配置列表

  1. 在左边的面板中选择 Widget Options

  2. 设置 Display Products 为下值之一：

    All Products

    列表中显示最近添加的产品

    New Products

    只显示标记为新的的产品，当前时间在产品是否是新的的时间范围内的才显示，所以如果这样设置，可能列表是空的

  3. 完成剩下的信息：
    a. 要给多页的情况提供导航，设置 Display Page Control 为 Yes
    b. 设置 Number of Products to Display 为新产品的数量，默认是 10 个
    c. 在 Cache Lifetime (Seconds) 中输入你希望多久刷讯一次列表，默认是 86400 秒，也就是 24 小时

  4. 完成后，点击 Save

  5. 按照提示刷新缓存。

第五步：预览

  1. 后台侧边栏，点击 Content，在 Elements 下，选择 Pages
  2. 找到新产品列表显示的页面，进行预览。
